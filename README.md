# Employee Management System

A simple Employee Management System built using **Python**, **SQLite**, and **SQLAlchemy**.

## Features

* Add Employee
* View Employees
* Menu-driven interface
* SQLite database
* SQLAlchemy ORM

## Technologies Used

* Python
* SQLAlchemy
* SQLite

## Installation

```bash
git clone https://github.com/keshavkaushik0509-boop/employee-management-system-sqlalchemy.git
cd employee-management-system-sqlalchemy
pip install sqlalchemy
python main.py
```
